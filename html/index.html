Projet LSSTack Documentation

L'objectif de ce site est de stocker toute la documentation est les données importante du projet que j'ai pu effectuer au LPC. Celui-ci a pour but d'étudier la qualité des données générées lors de la DC-2013.

Méthodes d'évaluation

Lors de ce projet différentes méthodes pour évaluer la qualité des données générée par le Stack furent utilisées. La comparaison à un jeu de données déjà existant, c'est le rôle de la partie compareSrc et l'étude de la stabilité temporelle des données qui utilise le même moteur de calcul. La quantité de données nécessaire à ces études, et celles générées sont importantes, ainsi une partie du projet est consacré à l'étude d'une base de données pour stocker ceci : stockSrc.

compareSrc

Cette partie est consacrée à la documentation du programme C++ permettant la comparaison d'un jeu de données à un autre. L'algorithmie utilisée est celle de la recherche de plus proche voisin. Différents paramètres sont ajoutés pour effectuer soit la comparaison des données du Stack avec SDSS, ou la comparaison temporelle des données ou effectuer l'association inter-filtre des données du Stack.
